## 5.3 Criação da Faixa Etária

Foi criada uma função personalizada:

```python
def faixa_etaria(idade):
    if idade < 18:
        return "Menor de idade"
    elif idade < 30:
        return "Jovem"
    elif idade < 60:
        return "Adulto"
    else:
        return "Senior"
```

### Objetivo

Classificar os usuários em categorias de idade.

### Categorias criadas

| Faixa          | Condição                    |
| -------------- | --------------------------- |
| Menor de idade | idade menor que 18          |
| Jovem          | idade entre 18 e 29         |
| Adulto         | idade entre 30 e 59         |
| Senior         | idade igual ou maior que 60 |

Essa categorização facilitou análises segmentadas.
